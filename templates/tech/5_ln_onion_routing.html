Every Lightning IOU moves through multiple nodes (aka servers) on its way to its final recipient. For each node in its path another layer of encryption is added. When a node receives a Lightning IOU it is able to decrypt one, and only one of these layers before sending it on to the next node.

This is a secure communication technology used in other software like Tor.

Onion routing is a part of the Lightning network and helps protect every transaction on it.

However keep in mind that no single piece of technology is a magic solution to all issues and privacy on Lightning can and will be improved in a variety of ways.
